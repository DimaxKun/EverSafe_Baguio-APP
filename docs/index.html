About the App

Built for the People of Baguio City

EverSafe Baguio is a mobile disaster preparedness application designed specifically for residents of Baguio City, Philippines. It delivers real-time weather monitoring, flood and landslide risk assessments, evacuation center navigation, and AI-powered safety guidance — all in one place.

With Baguio's mountainous terrain and susceptibility to typhoons and heavy rainfall, having instant access to safety information can be life-saving. EverSafe Baguio bridges the gap between residents and critical disaster response resources.

Architecture

How It's Structured

Clean MVVM architecture with a clear separation of concerns.

UI Layer
  • MainActivitySplash / entry point
  • HomeFragment host & notifications
  • DashboardFragmentWeather overview & risk popup
  • WarningFragmentForecast & barangay risk levels
  • LocationFragmentGoogle Maps + evacuation centers
  • TipsFragmentSafety tips
  • NotifFragmentNotification history
  • ChatAIGemini AI chat dialog
Data Layer
  • AppDatabaseRoom database instance
  • EvacuationCenterEntity model
  • EvacuationCenterDaoDatabase access object
  • EvacuationCenterRepositoryRepository pattern
Utilities
  • FloodLandslideRiskCalculatorRisk scoring logic
  • SharedWeatherViewModelLiveData weather state
  • SidebarManagerNavigation drawer
  • WeatherApiServiceRetrofit API interface
  • MessageAdapterChat RecyclerView adapter

Getting Started

Up and Running in Minutes

01
Prerequisites
  • Android Studio Hedgehog+
  • Android device / emulator API 26+
  • Google Maps API key
  • OpenWeatherMap API key
  • Google Gemini API key
02
Clone & Configure
Terminal
git clone https://github.com/your-username/eversafe-baguio.git
cd eversafe-baguio

Then add your API keys in AndroidManifest.xml, DashboardFragment.java, and ChatAI.java.

03
Build & Run
Terminal
./gradlew assembleDebug

Or open in Android Studio and click Run ▶. The app will install on your connected device or emulator.

Permissions

What We Ask For & Why

EverSafe Baguio only requests permissions that are essential to its core functionality.

ACCESS_FINE_LOCATION
Find the nearest evacuation center to your exact position
ACCESS_COARSE_LOCATION
Approximate location fallback when fine location is unavailable
INTERNET
Fetch weather data and communicate with AI APIs
POST_NOTIFICATIONS
Send disaster alert push notifications
CALL_PHONE
One-tap dialing to emergency services
FOREGROUND_SERVICE
Background weather monitoring and alerts
